# Simple CRUD application
CRUD frontend application with mocked data.
(WIP) It's possible to view list of existing items, overview details of selected item, edit or delete existing item, create new item. 
 
## Mocks
REST API used for application is provided by Mockoon. Configuration files can be found in folder /mocks/.
Mockoon doesn't allow to mock data exchange and used here as mocked data provider and emulation of success and unsuccess scenarious.
